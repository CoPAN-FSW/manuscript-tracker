# Manuscript Tracker

A simple, privacy-focused web tool for tracking manuscript submissions, reviews, and publication status across your research group.

## âœ¨ Features

- Track manuscript status (draft, submitted, under review, revising, accepted, rejected, published)
- Add notes and tags to each manuscript
- Automatic follow-up reminders for submissions waiting >30 days (review/submitted) or >14 days (revising)
- Search and sort manuscripts
- Export data to CSV (for Excel/Word)
- **All data stored locally in your browser** - nothing sent to servers
